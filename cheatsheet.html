Basic HTML

Tag Description
<!DOCTYPE> Defines the document type.
<html> Defines a document as HTML.
<head> Contains the information for the document.
<title> Names the page, may so up in the browser tab.
<body> Contains the content that users will see.
<h1> - <h6> Headers starting from most to least important.
<p> Defines a paragraph.
<br> Creates a line break in text.
<hr> Creates a bar to seperate content.
<--...--> Allows you to make notes in your code.
